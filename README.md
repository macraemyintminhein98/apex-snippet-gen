# Apex Snippet Gen

**Apex Snippet Gen** is an AI-powered web application designed to help junior developers, students, and experienced coders quickly generate common code snippets. Save time on repetitive tasks and focus on what matters – building great software.

## Features
- Instantly generate boilerplate code snippets.
- Supports various programming languages and frameworks (e.g., React, Python, CSS).
- Simple and intuitive user interface.

## How to Use
1. Enter a description of the snippet you need (e.g., "React useState boilerplate").
2. Click "Generate Snippet".
3. Copy the generated code and use it in your project.

## Getting Started
To run this project locally, clone the repository and install dependencies:

bash
git clone https://github.com/your-repo/apex-snippet-gen.git
cd apex-snippet-gen
npm install
npm run dev
